- 旅行期間：2027/11/17（三）～2027/11/21（日）
- 旅遊型態：五天四夜大阪自由行

- 核心定位：Luxury Slow Travel

- 航班：去程 EVA Air BR178，回程 EVA Air BR129；2027/11 官方班表與票價待確認。

- Day 3 交通主方案：電車＋短程計程車；KKday #133661 10 小時包車僅保留為雨天、疲勞或大量購物備案。

- 餐飲方向：好吃優先，是否米其林不重要。
