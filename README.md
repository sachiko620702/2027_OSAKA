# Web Publish Area

這個資料夾是網站發佈區，預設輸出檔為 `index.html`。

正式 GitHub Pages 網址：

- `https://sachiko620702.github.io/2027_HAL_Noordam/`

用途：

- 放 GitHub Pages 要公開的頁面
- 未來新增的網頁也放在這裡
- 不放 SQLite、種子資料、長篇說明或附件

更新方式：

1. 修改 `data/seed_data.py` 或 `data/export_site.py`
2. 執行 `python3 data/export_site.py`
3. 以 `web/index.html` 作為 GitHub Pages 發佈根目錄
