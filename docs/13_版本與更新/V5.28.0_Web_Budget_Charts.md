# V5.28.0｜Web 預算圖表化

Date: 2026-07-07  
Version: V5.28.0  
Status: Completed

---

## 1. 修改內容

將 `web/index.html` 的預算區塊改成「圖表 + 明細表」雙層呈現：

- 新增預算結構堆疊長條圖，顯示交通、住宿、購物、飲食、USJ、其他的比例。
- 新增建議版、節制版與安全版的比較圖，讓預算區間更直觀。
- 保留原本的預算明細表，供核對與追蹤使用。

---

## 2. 修改原因

使用者希望在 web 頁面上把預算用圖表呈現，讓顧客能更快看懂整體花費結構與安全上限。
