# V5.60.0 Web Multi-page Threads Style Redesign

Date: 2026-07-14
Status: Completed
Type: Website presentation refresh / no itinerary or budget decision change

## 修改目的

使用者希望參考 Threads 影片的節奏，並且不再侷限單頁輸出。V5.60.0 將 `web/` 改成多頁式旅行冊結構，讓首頁只做封面與導覽，其他內容拆成獨立章節頁。

## 修改範圍

| 檔案 | 修改內容 |
|---|---|
| `web/index.html` | 改成入口首頁，保留深色封面與章節導覽，連到行程、預算、行前與餐飲分頁。 |
| `web/itinerary.html` | 新增行程頁，集中呈現 Day 1 至 Day 5 的正式節奏與固定假設。 |
| `web/budget.html` | 新增預算頁，集中呈現建議版、節制版、安全版與各分類預算。 |
| `web/prep.html` | 新增行前頁，集中呈現行李、文件、天氣與關鍵日期。 |
| `web/dining.html` | 新增餐飲頁，集中呈現正式晚餐與 VIP 用餐節奏。 |
| `web/styles.css` | 補上多頁導覽的 current-page 樣式與章節卡片連結樣式。 |
| `README.md` | 更新目前主控版本為 V5.60.0。 |
| `CHANGELOG.md` | 新增 V5.60.0 變更紀錄。 |

## 設計方向

- 首頁保留 Threads 風格封面感，但不再把所有內容塞成單頁長卷軸。
- 以四個章節頁拆分資訊，讓讀者可以先看封面與目錄，再進入細節。
- 內容仍沿用同一套 database 與預算基準，不建立平行方案。
