# V5.62.0 Web Documents Manager Refresh

Date: 2026-07-14
Status: Completed
Type: Website presentation refresh / no itinerary or budget decision change

## 修改目的

使用者希望行李清單／文件區看起來更像文件管理器，並具備預覽、下載、上傳的互動。V5.62.0 將 `web/prep.html` 的上半部改成資料夾式文件卡片介面。

## 修改範圍

| 檔案 | 修改內容 |
|---|---|
| `web/prep.html` | 新增文件管理器區塊，包含上傳、預覽、下載、刪除與重新整理操作。 |
| `web/prep.js` | 新增文件上傳與預覽邏輯，支援圖片與 PDF 檢視。 |
| `web/styles.css` | 新增文件管理器、文件卡片、預覽視窗與按鈕樣式。 |
| `README.md` | 更新目前主控版本為 V5.62.0。 |
| `CHANGELOG.md` | 新增 V5.62.0 變更紀錄。 |

## 設計方向

- 參考使用者提供的文件頁 UI，改成較輕、較白、較像檔案管理器的視覺。
- 文件條目以資料夾卡片呈現，保留「上傳 / 預覽 / 下載」的常見操作。
- 行李與天氣等旅行資訊仍保留在同頁，但視覺重點先落在文件區。
