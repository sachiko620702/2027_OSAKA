# V5.61.0 Web Threads Film Rhythm Refresh

Date: 2026-07-14

Type: Website presentation refresh / no itinerary or budget decision change

## 修改目的

使用者希望多頁版再更接近 Threads 影片節奏，並且讓封面感更強。V5.61.0 進一步把首頁做成更像影片開場的視覺節拍，讓章節切換更有分鏡感。

## 修改範圍

| 檔案 | 修改內容 |
|---|---|
| `web/index.html` | 加強封面首屏、章節脊椎與 story-beat 區塊，讓首頁更像影片開場。 |
| `web/styles.css` | 新增封面 cue、章節 spine 與 story-beat 的視覺樣式，並補強小螢幕排版。 |

## 設計方向

- 首頁封面更強調「第一格畫面」的存在感。
- 章節不只是連結，而是像影片分鏡一樣有節拍。
- 保留多頁式結構，但讓封面與目錄的過場更有故事感。
