# V5.52.0 Website Visual Storytelling Refresh

Date: 2026-07-08  
Status: Completed  
Type: Website presentation refresh / no itinerary or budget decision change

## 修改目的

使用者指出目前網站缺少圖文並茂與旅行社宣傳頁面的感覺。V5.52.0 針對 `web/` 進行視覺敘事與版面強化，讓首頁更像顧客可閱讀的旅行提案，而不是內部備忘錄。

## 修改範圍

| 檔案 | 修改內容 |
|---|---|
| `web/index.html` | 重構首頁敘事，新增旅行社風格 hero、視覺亮點區、圖文式每日行程卡、餐飲視覺卡與新版頁尾版本標記。 |
| `web/styles.css` | 重寫視覺樣式，新增海報式首屏、照片牆感卡片、行程圖像卡、深色精品旅行配色、響應式版面與保留預算 JS 所需樣式。 |
| `docs/14_自動同步狀態/每日同步狀態.md` | 補入 V5.52.0 網站視覺更新狀態。 |
| `README.md` | 更新目前主控版本為 V5.52.0 Website Visual Storytelling Refresh，並標明正式行程與預算仍沿用 V5.51.0 主控基準。 |

## 設計方向

- 首屏改為旅行社提案式宣傳語：大阪五天四夜慢旅行提案。
- 新增右側旅程視覺摘要：Hotel Lounge、USJ、Shopping、Sushi 等圖像感卡片。
- 新增 `Visual Story` 區塊，將飯店、USJ、木津市場、日本橋、心齋橋與餐飲做成圖文亮點卡。
- 每日行程卡改成上圖下文的 editorial card，讓 Day 1–Day 5 一眼更有旅行畫面。
- 餐飲區新增視覺卡片，強化千房、肉NOASATSU、NOKA Roast & Grill 的顧客閱讀感。
- 不使用外部照片，避免授權與來源問題；目前以 CSS 漸層視覺模擬宣傳頁照片感。
