# V5.3.0 - Daily Checks and Status Reporting

Date: 2026-07-06  
Version: V5.3.0  
Status: Completed

---

## 修改內容

新增每日自動化監測流程：

- `scripts/daily_maintenance.py` 會在同步固定資料後，對設定好的來源頁面執行每日價格與開賣狀態檢查。
- `database/daily_checks.json` 集中管理可自動監測的來源與手動提醒項目。
- `database/daily_status.json` 保存最新一次每日檢查結果，方便 Git 同步與後續比對。

---

## 修改原因

專案已經有固定資料同步需求，下一步是把每日變動項目也收進自動化流程，讓價格、開賣狀態與必要的人工復核都能透過同一條排程產出。
