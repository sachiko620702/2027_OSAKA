Version: V5.25.0  
Last updated: 2026-07-06  
Status: 正式行程  
Scope: Day 3 不去京都，改為大阪市內 Luxury Slow Travel

---

## 1. 本版決策

使用者明確指示：

> 晚餐吃壽司，覆蓋正式行程，並更新所需更新章節。

因此 Day 3 正式行程改為：

> 木津市場早午餐 → 難波／日本橋動漫模型周邊 → 飯店休息＋行政酒廊下午茶 → 大阪市內壽司晚餐

京都版 Day 3、京 鰻和 本店、天ぷら 京星與京都甜點路線改為備案／歷史版本，不再是 Day 3 主方案。

## 4. 交通原則

| 項目 | 原則 |
|---|---|
| 木津市場 → 日本橋 | 大阪市內地鐵／短程計程車，依當天體力與天氣調整。 |
| 日本橋 → 難波／道頓堀／心齋橋 | 步行或短程地鐵。 |
| 午餐後 → InterContinental Osaka | 地鐵至梅田後步行，或購物多時直接計程車。 |
| 晚餐 | 優先選梅田／北新地／福島／中之島／難波北側，避免遠距離移動。 |
